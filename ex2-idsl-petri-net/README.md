## Static Semantics

- A petri net
  - has a name (non-empty)
  - consists of at least one place

- A place
  - has a name (unique, non-empty)
  - has an initial amount of tokens (>= 0)
  - has at least one ingoing or one outgoing transaction

- A transaction
  - has a name (non-empty)
  - has transaction costs (>=0)
